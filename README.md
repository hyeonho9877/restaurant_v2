======
2021년 소프트웨어공학 팀프로젝트로 구현한 레스토랑 예약 시스템   
기존 Java 기반 GUI 윈도우 어플리케이션에서 웹 서비스로 고도화 작업 수행   
**백엔드** : Spring Boot + JSP   
**프론트엔드** : HTML5 + CSS + JS + JQUERY


기능
---
1. 회원 가입 및 로그인/아웃   
  1-1. 아이디 및 비밀번호 찾기   
  1-2. 비밀번호 수정   
2. 예약 및 취소   
  2-1. 인원 선택   
  2-2. 날짜 선택   
  2-3. 메뉴 선택   
  2-4. 좌석 선택   
4. 예약 수정   
5. 관리자 페이지   
  5-1. 예약 리스트 확인   
  5-2. 메뉴 리스트 수정   
  5-2. 예약 테이블 변경   
  5-3. 테이블 수 변경   
  5-4. 통계 자료 확인
